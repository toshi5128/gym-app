# ATLAS カラーパレット（ひかり不動産への共有用）

「温かいアイボリー × 金属ゴールド × チャコール文字」の配色仕様。
ひかり不動産のブランドカラー（黄色〜オレンジ系）と相性が良く、そのまま寄せられます。

---

## 1. ブランド3色（最低限これだけでOK）

| 役割 | HEX | 説明 |
|---|---|---|
| 背景（アイボリー） | `#f1f0ec` | 少し温かい白。青白い白にしないのが上質感の決め手 |
| 文字（チャコール） | `#1c1a17` | 黒に近い焦げ茶。真っ黒より柔らかい |
| ブランドゴールド | `#a8842f` | 白地の文字・アイコン・差し色。コントラスト確保した濃いめの金 |

## 3. “安っぽい金”にしないための3ルール（重要）

1. **金のベタ塗りは避ける** — 塗りはグラデ、ハイライト（リング等）は3stopの金属グラデにする。平らな金は安く見える。
2. **白地の金文字は濃いめ `#a8842f`** を使う。明るい金（`#e3c777` など）は“面のハイライト専用”で、文字には使わない（コントラスト不足で読みにくくなる）。
3. **背景は青白い白でなく、必ず少し温かい白 `#f1f0ec`** に。冷たい白＋金は安っぽく、温かい白＋金で高級感が出る。

## 4. そのまま貼れるCSS変数

```css
:root{
  --bg:#f1f0ec;
  --surface:#ffffff;
  --surface-2:#f6f4ef;
  --surface-3:#ece8df;
  --hair:rgba(28,24,16,.13);
  --txt:#1c1a17;
  --txt-2:#5a5347;
  --txt-3:#988f7e;
  --gold:#a8842f;        /* 差し色・白地の金文字 */
  --gold-deep:#876a23;   /* グラデ終端 */
  --gold-dim:rgba(168,132,47,.13); /* 淡い面・選択状態 */
  --shadow:0 4px 14px -6px rgba(60,48,20,.10), 0 20px 44px -22px rgba(60,48,20,.15);
}
/* ボタン例 */
.btn-primary{ background:linear-gradient(160deg,var(--gold),var(--gold-deep)); color:#fff; }
/* 金属ハイライト（リング等）例 */
/* linear-gradient: #876a23 → #bd9740 → #e3c777 */
```

---

## 5. ひかり不動産に寄せる場合の微調整

- ひかりのロゴが**もっと明るい黄色/オレンジ**なら、`--gold` をロゴの色相に少し寄せても可（例：`#b5892b`〜`#c79a2e`）。ただし**白地の文字に使う金は彩度を上げすぎない**こと（読みにくくなる）。明るい色はボタン塗りや帯のハイライトに回す。
- ベース（背景アイボリー・チャコール文字・影）は変えずに、**差し色の金だけブランドのトーンに合わせる**のが、上質感を保ったまま寄せるコツ。
