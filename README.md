# ATLAS — 筋トレ記録アプリ

個人用の筋トレ記録 PWA。スマホのホーム画面に追加して、ネイティブアプリのように使える。

- **構成**: プレーンな HTML/CSS/JS の1ファイル（`index.html`）。ビルド不要。
- **データ**: 端末内に保存（localStorage）。サーバー不要。
- **バックアップ**: アプリ内「設定 → データ」から JSON で書き出し／読み込み。
- **PWA**: `manifest.json` + `sw.js`（オフライン起動）+ `icon-192/512.png`。

## ローカルで開く
`index.html` をブラウザで開くだけ。
（※サービスワーカー＝オフライン機能は https もしくは GitHub Pages 公開時のみ有効。`file://` では無効。）

## 公開（GitHub Pages）
このリポジトリの **Settings → Pages → Deploy from a branch → `main` / `(root)`** を選ぶと、
`https://<ユーザー名>.github.io/gym-app/` で配信される。push するたび自動反映。

## 主な機能
- 4分割ローテ（胸→背中→肩→足）の自動サジェスト＆メニュー
- 重量×回数のセット記録、前回比較・自己ベスト（PR）演出
- インターバルタイマー（記録後に自動開始／音・振動）
- 履歴・種目別の推定1RM推移グラフ・体重ログ
